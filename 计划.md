# Python面试在线答题系统 - 开发计划

## 项目概述
基于Django + Vue 3的前后端分离项目，接入小米MiMo大模型API，实现在线出题和在线判题功能。

## 技术栈
- 前端：Vue 3 + Element Plus + Axios
- 后端：Django + Django REST Framework
- 数据库：SQLite（开发）/ PostgreSQL（生产）
- AI：小米MiMo大模型API

## 功能模块

### 1. 用户模块
- 用户注册/登录
- JWT认证
- 用户信息管理

### 2. 题库模块
- 题目分类（选择题、填空题、编程题）
- 题目CRUD操作
- 题目难度标签

### 3. AI出题模块
- 调用MiMo API生成题目
- 支持指定主题、难度
- 题目存入数据库

### 4. 答题模块
- 在线答题界面
- 计时功能
- 提交答案

### 5. AI判题模块
- 调用MiMo API评判答案
- 生成评分和解析
- 存储答题记录

### 6. 成绩模块
- 答题历史记录
- 成绩统计分析
- 错题本功能

## 数据库设计

### 用户表 (User)
- id, username, password, email, created_at

### 题目表 (Question)
- id, type, content, options, answer, difficulty, category, created_at

### 答题记录表 (ExamRecord)
- id, user_id, question_id, user_answer, score, ai_feedback, created_at

### 试卷表 (Exam)
- id, user_id, title, questions, total_score, status, created_at

## API设计

### 用户接口
- POST /api/users/register/
- POST /api/users/login/
- GET  /api/users/profile/

### 题目接口
- GET  /api/questions/
- POST /api/questions/
- GET  /api/questions/{id}/

### AI接口
- POST /api/ai/generate/      # 生成题目
- POST /api/ai/evaluate/      # 评判答案

### 答题接口
- POST /api/exams/create/
- POST /api/exams/submit/
- GET  /api/exams/records/
